# 📊 SmartFit AI — Sample Evaluation & Training Dataset

To maintain a lightweight, fast-cloning GitHub repository while providing full transparency into our computer vision and deep learning training data, we have structured our repository dataset as follows:

## **📁 Contents**
Instead of uploading the entire **900 MB+ raw video dataset** (which exceeds GitHub's file size limits and slows down repository cloning), this dataset includes:

1. **`raw_frames/` (12 Exercise Classes — 10+ Sample Frames Each):**  
   Contains **10+ benchmark visual image frames across all 12 distinct workout activities** (Push-ups, Squats, Planks, Bicep Curls, Lunges, Jumping Jacks, Deadlifts, Crunches, Shoulder Presses, etc.). These frames demonstrate the exact RGB visual inputs processed by our OpenCV and MediaPipe pose estimation pipeline.

2. **`pose_landmarks/` (Extracted 3D Keypoints):**  
   Contains the complete set of extracted 3D skeleton joint coordinates (`x, y, z, visibility` across 33 landmarks) generated by MediaPipe Pose. These mathematical representations remove background noise and isolate human biomechanics.

3. **`sequences/` (Temporal Time-Series Data):**  
   Contains the formatted sequence arrays fed directly into our custom **1D-CNN + LSTM neural network** for time-series action recognition and repetition state classification.

## **⚙️ Engineering Rationale & Training Workflow**
In our data preprocessing pipeline, raw exercise videos were first converted into thousands of individual RGB image frames (**`raw_frames/`**). Next, MediaPipe Pose processed each frame to extract 3D skeleton coordinates (**`pose_landmarks/`**) and formatted them into time-series sequence arrays (**`sequences/`**) to train our custom **1D-CNN + LSTM neural network**.

To keep this GitHub repository fast and lightweight without exceeding storage limits, we have included the **complete mathematical training datasets (`pose_landmarks/` and `sequences/`)** alongside **representative sample image frames (`raw_frames/`)**. This allows any developer or data scientist to fully inspect, evaluate, and retrain our deep learning architecture without downloading 900 MB+ of raw image files!
